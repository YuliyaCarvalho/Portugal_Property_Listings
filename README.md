# Real Estate Market Analysis - Portugal
### An exploratory data analysis of real estate listings across Portugal, examining property features, pricing patterns, and geographic distributions. This project analyzes asking prices and property characteristics to uncover insights about the Portuguese housing market.

Dataset: Real Estate Listings in Portugal (Kaggle) - 17MB CSV containing comprehensive property features and pricing data

Technologies: Python, Pandas, Matplotlib/Seaborn, Jupyter Notebook (Google Colab)

Key Analysis Areas:

* Price distribution and trends across Portuguese regions
* Property features impact on asking prices
* Geographic patterns in real estate listings
* Market segmentation by property type and location
